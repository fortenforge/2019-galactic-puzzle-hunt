Hints: Colors

Getting started:
Focus on the hex codes of the colors in each image. It might be useful to write the hex characters in a 6x6 grid. Each image can be treated separately at this point.
Subpuzzle hints:
  • 1: This subpuzzle is based on a popular puzzle game that comes with Windows operating systems.
  • 2: Each cell is related to the two cells above it.
  • 3: Each row can be treated separately. Does 2,3,5,7,... or “14159” sound familiar?
  • 4: Try reading down columns
  • 5: Unlike the other subpuzzles, this one actually represents real colors. Further hint: The colors are specifically Pantone colors.
  • 6: This one represents a common logic puzzle type.
Stuck on extraction:
The positions of the replaced characters within the 6x6 grids are important. In particular, they are all distinct.
Further hint:
The answers to the clues are X11 web colors.
